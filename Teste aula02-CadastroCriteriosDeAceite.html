CENÁRIO DE TESTE 01: CADASTRO DE USUÁRIO. REQUISITOS: ACESSAR O SISTEMA WWW.EEE.COM.BR; CRITERIOS DE ACEITE: • CASO O CADASTRO SEJA REALIZADO COM SUCESSO, RETORNAR PARA TELA DE LOGIN. • SENHA DE NO MINIMO 8 CARACTERES. DADO QUE O USUÁRIO TENHA ACESSADO O SISTEMA; E SELECIONADO O BOTÃO DE CADASTRAR USUÁRIO; E PREENCHER OS CAMPOS OBRIGATÓRIOS COM DADOS VÁLIDOS; QUANDO PREENCHER TODOS OS CAMPOS E SELECIONAR O BOTÃO “CADASTRAR”; ENTÃO O SISTEMA DEVERÁ RETORNAR UMA MENSAGEM DE SUCESSO E RETORNAR O USUÁRIO PARA A PÁGINA DE LOGIN. CENÁRIO DE TESTE 02: CADASTRO DE USUÁRIO SEM PREENCHIMENTO DOS CAMPOS. REQUISITOS: ACESSAR O SISTEMA WWW.EEE.COM.BR; CRITÉRIOS DE ACEITE: • RETORNAR MENSAGEM DE ERRO AO EFETUAR TENTATIVA DE CADASTRO SEM PREENCHIMENTO DOS CAMPOS. • INFORMAR A BAIXO DOS CAMPOS (LABEL) QUE QUE OS CAMPOS OBRIGATÓRIOS PRECISAM SER PREENCHIDOS PARA CONTINUAR. DADO QUE O USUÁRIO TENHA ACESSADO O SISTEMA; E SELECIONADO O BOTÃO DE CADASTRAR USUÁRIO; E PROSSEGUIR O CADASTRO SEM O PREENCHIMENTO DE UM DOS CAMPOS OBRIGATÓRIOS; QUANDO O USUÁRIO AVANÇAR PRESSIONANDO O BOTÃO “CADASTRAR”; ENTÃO O SISTEMA DEVERÁ RETORNAR UMA MENSAGEM DE “ERRO” INFORMANDO QUE NÃO FOI EFETUADO O CADASTRO.
